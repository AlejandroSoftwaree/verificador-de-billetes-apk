# Detector-de-Billetes-Serie-B
**Funcionalidades**
*   **Reconocimiento Automático**: Implementación de `google_ml_kit` de lectura de texto en tiempo real para capturar números de serie mediante la cámara.
*   **Selección de Denominaciones**: Interfaz intuitiva para elegir el valor del billete antes del escaneo, optimizando la precisión de la validación.
*   **Ingreso Manual**: Formulario alternativo con validación inteligente para prevenir errores de escritura y asegurar el formato correcto.
*   **Uso sin Conexión**: La aplicación no necesita de conexión a internet para funcionar (validación local de base de datos).
*   **Panel de Resultados**: Visualización clara y detallada que indica si el billete se encuentra dentro de un rango válido o si presenta alguna observación por serie ilegal.
*   **Canales de Ayuda**: Integración directa con soporte técnico a través de WhatsApp y correo electrónico para reportes rápidos desde la pantalla de contacto.
**Compatibilidad Android**
*   Soporte para dispositivos **Android 5.0 (API 21)** en adelante.
*   Compatible con las últimas versiones de Android (API 34/35).
